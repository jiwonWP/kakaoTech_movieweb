# Kookiflix 🎬

TMDB API를 활용한 영화 검색 웹사이트입니다.  
외부 프레임워크나 라이브러리 없이 순수 JS만으로 구현되었으며
사용자는 인기 영화 목록을 확인하고, 영화 제목으로 검색하며, 각 영화의 상세 정보를 모달로 확인할 수 있습니다.

## 🔧 주요 기능

- 🎥 TMDB API 연동 (인기 영화 / 검색 기능 / 상세 정보)
- 🔍 영화 검색 기능 (Enter 키, 버튼 클릭 지원)
- 📦 모듈화된 코드 구조
- 💬 영화 상세 정보 모달 구현
- 🌐 반응형 디자인 적용
